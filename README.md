# MUSIC RECOMMENDATION SYSTEM
A content-based filtering music recommendation web application built with python and streamlit. It suggest songs based on lyrics similarity using Machine Learning (Cosine Similarity) and fetches real time album cover using the Spotify API
# Features: 
  Content-Based Filtering: Recommends songs similar to your  selection based on lyrics and text data.
  Spotify Integration: Automatically fetches high-quality album art for every recommendation.
  Interactive UI: Clean and simple web interface powered by Streamlit.
  Top 5 Recommendations: Instantly displays the top 5 most similar songs with posters.
# Tech Stack
  Frontend: Streamlit
  Backend:Python
  Machine Learning: Scikit-Learn (CountVectorizer, Cosine Similarity)
  Data Handling: Pandas, Pickle
  API: Spotipy (Spotify Web API)
